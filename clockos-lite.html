What ClockOS Lite includes

The essentials.
Nothing extra.

ClockOS Lite is intentionally stripped down — fast, lightweight, and focused on the two things every display needs: show the time and push the alert.

🕐
Clock + schedule display
Time, date, current period, and next bell — always on screen. Syncs with the ClockOS server automatically.
🚨
Full alert system
All 8 ClockOS alert types — Hold, Secure, Lockdown, Evacuate, Shelter, Fire, Standby, All Clear. Full-screen color alerts, received instantly from the ClockOS dashboard.
📢
Announcements
Non-urgent announcements pushed from the admin dashboard display on screen. Auto-dismiss after a set duration.
🏫
School branding
School name, colors, and bell schedule configurable per device via the ClockOS admin dashboard.
📡
Remote management
Devices appear in the ClockOS admin dashboard alongside Halo and Beacon+ units. Update, reboot, and configure remotely.
⚡
Lightweight footprint
Designed for lower-spec hardware. Runs on Raspberry Pi 2GB RAM and most Android-based commercial displays.
